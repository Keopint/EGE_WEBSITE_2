Здравствуй, пользователь! Проект Hampters EGE представляет из себя сайт 
для подготовки школьников к Единому Государственному Экзамену. Мы предоставляем
огромный список инструментов: банк заданий ЕГЭ, возможность добавления своих заданий, 
варианты, составление вариантов, курсы, создание курсов, добавление и просмотр 
презентаций, статистика по решению задач и многое другое!

Инструкция по установке: 
Скачайте архив проекта с github: https://github.com/Keopint/EGE_WEBSITE_2

Разархивируйте файл проекта
Запутите файл проекта с помощью VS studio или pycharm
Установите библиотеки с помощью команды pip install -r requirements.txt

Запуск сайта:
Запустите файл app.py

Сайт будет доступен в браузере по адресу: http://127.0.0.2:8080S
